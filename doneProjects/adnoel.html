Web application de gestion de commandes

React - Nodejs / Express - MongoDB

Page principale

Formulaire d'enregistrement des commandes

Les utilisateurs rentrent les informations de chaque commandes pour les envoyer dans la base de données. L'utilisateur doit être connecté pour entrer une commande.

Partie administateur

L'admin récupère les commandes dans un tableaux

Celui-ci peut télécharger les commandes en pdf. Il peut supprimer ou modifier une commande existante.

Les librairies utilisées

moment - jsonwebtoken - material icon - axios - redux - sweet-alert - jspdf

bcrypt - body-parser - cors - dotenv - mongoose - nodemon
